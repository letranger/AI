1. 前言：為什麼要用自己的資料？

之前的課程中，我們都是用現成的資料集（MNIST 手寫數字、CIFAR-10 等）來訓練模型。這些資料集很方便，但在真實世界中，你遇到的問題往往沒有現成的資料集可用。

例如：

  • 你想訓練一個模型分辨學校裡的貓和狗
  • 你想讓 AI 辨識不同品種的花
  • 你想建一個系統分辨垃圾有沒有做好回收分類

這時候，你就需要 *自己準備資料、自己訓練模型*。

這堂課，我們會用「貓 vs 狗」的圖片分類任務，從頭走一遍完整的流程：

  1. 準備資料（蒐集圖片、整理資料夾）
  2. 讀取圖片並轉成模型能吃的格式
  3. 建立 CNN 模型
  4. 訓練與評估
  5. 資料擴增（Data Augmentation）提升效果

2. 步驟一：準備資料

2.1. 資料夾結構

我們需要把圖片整理成以下結構——每個類別放一個資料夾：

cats_and_dogs/
├── cats/
│   ├── cat.0.jpg
│   ├── cat.1.jpg
│   ├── ...
│   └── cat.99.jpg
└── dogs/
    ├── dog.0.jpg
    ├── dog.1.jpg
    ├── ...
    └── dog.99.jpg
  • 兩種動物各 100 張圖片（你可以準備更多，越多越好）
  • 你可以點這裡下載範例圖片，解壓縮到 ~/Downloads/ 資料夾

準備自己的資料集小提醒：

  • 每個類別至少要 50 張以上（越多越好）
  • 圖片大小不用一樣，程式會自動調整
  • 避免放入損壞的圖檔
  • 不同類別的圖片數量盡量平衡

2.2. 讀取資料

 2: import cv2

 5: # 設定圖片資料夾路徑（請依你的實際路徑修改）
 6: img_dir = os.path.expanduser('~/Downloads/cats_and_dogs')
 7: cats_dir = os.path.join(img_dir, 'cats')
 8: dogs_dir = os.path.join(img_dir, 'dogs')
 9: 
10: cat_fnames = [f for f in os.listdir(cats_dir) if f.endswith(('.jpg', '.png', '.jpeg'))]
11: dog_fnames = [f for f in os.listdir(dogs_dir) if f.endswith(('.jpg', '.png', '.jpeg'))]
12: 
13: print(f"貓的圖片數量：{len(cat_fnames)}")

2.3. 將圖片讀入並轉成 numpy 陣列

所有圖片大小不一，但模型需要統一大小的輸入。我們把每張圖片都縮放到 128x128 像素（比 224x224 小，訓練更快，對小資料集來說也夠用了）。

 1: IMG_SIZE = 128  # 統一圖片大小
 2: 
 3: data = []    # 存放圖片
 4: target = []  # 存放標籤（0=貓, 1=狗）
 5: 
 6: # 讀取貓的圖片
 7: for fname in cat_fnames:
 8:     img = cv2.imread(os.path.join(cats_dir, fname))
 9:     if img is not None:
10:         img = cv2.resize(img, (IMG_SIZE, IMG_SIZE))  # 縮放
11:         img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)   # BGR → RGB
12:         data.append(img)
13:         target.append(0)  # 貓 = 0
14: 
15: # 讀取狗的圖片
16: for fname in dog_fnames:
17:     img = cv2.imread(os.path.join(dogs_dir, fname))
18:     if img is not None:
19:         img = cv2.resize(img, (IMG_SIZE, IMG_SIZE))
20:         img = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
21:         data.append(img)
22:         target.append(1)  # 狗 = 1
23: 
24: print(f"成功讀取 {len(data)} 張圖片")

3. 步驟二：資料預處理

3.1. 轉成 numpy 陣列並正規化

 1: # 轉成 numpy 陣列
 2: data = np.array(data, dtype=np.float32)
 3: target = np.array(target)
 4: 
 5: print(f"圖片資料形狀：{data.shape}")     # (N, 128, 128, 3)
 6: print(f"標籤資料形狀：{target.shape}")    # (N,)
 7: print(f"像素值範圍：{data.min()} ~ {data.max()}")
 8: 
 9: # 正規化：將像素值從 0~255 縮放到 0~1
10: data = data / 255.0
11: print(f"\n正規化後像素值範圍：{data.min():.1f} ~ {data.max():.1f}")

為什麼要正規化？

圖片的像素值範圍是 0~255，如果直接丟進模型，這麼大的數值會讓梯度爆炸，訓練效果很差。除以 255 後，所有值都落在 0~1 之間，模型學習起來更穩定。

如果你之前的模型準確度只有 50% 左右（等於隨機猜），很可能就是忘了做正規化！

3.2. 將標籤轉成 One-Hot Encoding

1: import keras
2: 
3: # One-hot encoding
4: target_onehot = keras.utils.to_categorical(target, num_classes=2)
5: print(f"原始標籤（前5個）：{target[:5]}")
6: print(f"One-hot 編碼：")
7: for i in range(5):
8:     label_name = "貓" if target[i] == 0 else "狗"
9:     print(f"  {label_name} → {target_onehot[i]}")

3.3. 分割訓練集與測試集

 1: from sklearn.model_selection import train_test_split
 2: 
 3: x_train, x_test, y_train, y_test = train_test_split(
 4:     data, target_onehot,
 5:     test_size=0.2,       # 20% 當測試資料
 6:     random_state=42,
 7:     stratify=target       # 確保訓練集和測試集的貓狗比例一致
 8: )
 9: 
10: print(f"訓練集：{x_train.shape[0]} 張")
11: print(f"測試集：{x_test.shape[0]} 張")
12: print(f"\n訓練集中的貓：{int(y_train[:, 0].sum())} 張")
13: print(f"訓練集中的狗：{int(y_train[:, 1].sum())} 張")

4. 步驟三：建立 CNN 模型

我們用一個簡單但有效的 CNN 架構。關鍵是：圖片越大，需要越多的卷積層來逐步提取特徵。

 1: from keras import layers, models
 2: 
 3: model = models.Sequential([
 4:     # 輸入層
 5:     layers.Input(shape=(IMG_SIZE, IMG_SIZE, 3)),
 6: 
 7:     # 第一組卷積：學習基礎特徵（邊緣、顏色）
 8:     layers.Conv2D(32, (3, 3), activation='relu'),
 9:     layers.MaxPooling2D((2, 2)),
10: 
11:     # 第二組卷積：學習進階特徵（紋理、形狀）
12:     layers.Conv2D(64, (3, 3), activation='relu'),
13:     layers.MaxPooling2D((2, 2)),
14: 
15:     # 第三組卷積：學習高階特徵（耳朵、鼻子）
16:     layers.Conv2D(128, (3, 3), activation='relu'),
17:     layers.MaxPooling2D((2, 2)),
18: 
19:     # 攤平並分類
20:     layers.Flatten(),
21:     layers.Dropout(0.5),              # 隨機關閉 50% 神經元，防止過度配適
22:     layers.Dense(128, activation='relu'),
23:     layers.Dense(2, activation='softmax')  # 2 個類別：貓、狗
24: ])
25: 
26: model.summary()
27: 
28: # 編譯模型
29: model.compile(
30:     optimizer='adam',
31:     loss='categorical_crossentropy',  # 搭配 one-hot 標籤
32:     metrics=['accuracy']
33: )

模型架構解讀：

  • 三層 =Conv2D + MaxPooling2D=：像是一個漏斗，把 128x128 的大圖逐步壓縮，同時提取越來越抽象的特徵
  • =Dropout(0.5)=：訓練時隨機「關掉」一半的神經元，迫使模型不依賴任何單一特徵，減少過度配適
  • =Dense(2, softmax)=：輸出兩個機率值，代表「是貓的機率」和「是狗的機率」

4:     epochs=20,

7. 進階：資料擴增（Data Augmentation）

當你的資料量很少（像我們只有 200 張），模型很容易過度配適——在訓練資料上表現很好，但碰到新圖片就不行了。

資料擴增 是解決這個問題的有效方法：對同一張圖片做旋轉、翻轉、縮放等變換，產生「新」的訓練資料。

 1: from keras.src.utils import image_utils
 2: 
 3: # 示範：對一張貓的圖片做各種變換
 4: sample_img = x_train[0]
 5: 
 6: fig, axes = plt.subplots(2, 4, figsize=(14, 7))
 7: axes[0, 0].imshow(sample_img)
 8: axes[0, 0].set_title('Original')
 9: 
10: # 水平翻轉
11: axes[0, 1].imshow(np.fliplr(sample_img))
12: axes[0, 1].set_title('Horizontal Flip')
13: 
14: # 垂直翻轉
15: axes[0, 2].imshow(np.flipud(sample_img))
16: axes[0, 2].set_title('Vertical Flip')
17: 
18: # 旋轉 90 度
19: axes[0, 3].imshow(np.rot90(sample_img))
20: axes[0, 3].set_title('Rotate 90')
21: 
22: # 亮度調整（變亮）
23: bright = np.clip(sample_img * 1.5, 0, 1)
24: axes[1, 0].imshow(bright)
25: axes[1, 0].set_title('Brighter')
26: 
27: # 亮度調整（變暗）
28: dark = np.clip(sample_img * 0.5, 0, 1)
29: axes[1, 1].imshow(dark)
30: axes[1, 1].set_title('Darker')
31: 
32: # 加入雜訊
33: noisy = np.clip(sample_img + np.random.normal(0, 0.05, sample_img.shape), 0, 1)
34: axes[1, 2].imshow(noisy)
35: axes[1, 2].set_title('Add Noise')
36: 
37: # 隨機裁切再縮放
38: h, w = sample_img.shape[:2]
39: crop_size = int(h * 0.8)
40: start_h = np.random.randint(0, h - crop_size)
41: start_w = np.random.randint(0, w - crop_size)
42: cropped = sample_img[start_h:start_h+crop_size, start_w:start_w+crop_size]
43: axes[1, 3].imshow(cropped)
44: axes[1, 3].set_title('Random Crop')
45: 
46: for ax in axes.flat:
47:     ax.axis('off')
48: 
49: plt.suptitle('Data Augmentation Examples', fontsize=14)
50: plt.tight_layout()
51: plt.savefig("images/mydata-augmentation.png", dpi=100)
52: plt.close()
53: print("資料擴增範例已儲存至 images/mydata-augmentation.png")
54: print("\n同一張圖片經過不同的變換，就能產生 8 種不同的訓練資料！")
55: print("如果原本有 200 張圖，每張做 8 種變換，就等於有 1600 張訓練資料。")

7.1. 使用 Keras 內建的資料擴增層

Keras 提供了方便的資料擴增層，可以直接加在模型中：

 1: # 建立帶有資料擴增的增強版模型
 2: model_aug = models.Sequential([
 3:     layers.Input(shape=(IMG_SIZE, IMG_SIZE, 3)),
 4: 
 5:     # 資料擴增層（只在訓練時生效，推論時自動關閉）
 6:     layers.RandomFlip("horizontal"),        # 隨機水平翻轉
 7:     layers.RandomRotation(0.1),             # 隨機旋轉 ±10%
 8:     layers.RandomZoom(0.1),                 # 隨機縮放 ±10%
 9: 
10:     # CNN 架構（和之前一樣）
11:     layers.Conv2D(32, (3, 3), activation='relu'),
12:     layers.MaxPooling2D((2, 2)),
13:     layers.Conv2D(64, (3, 3), activation='relu'),
14:     layers.MaxPooling2D((2, 2)),
15:     layers.Conv2D(128, (3, 3), activation='relu'),
16:     layers.MaxPooling2D((2, 2)),
17: 
18:     layers.Flatten(),
19:     layers.Dropout(0.5),
20:     layers.Dense(128, activation='relu'),
21:     layers.Dense(2, activation='softmax')
22: ])
23: 
24: model_aug.compile(
25:     optimizer='adam',
26:     loss='categorical_crossentropy',
27:     metrics=['accuracy']
28: )
29: 
30: print("增強版模型（含資料擴增層）：")
31: model_aug.summary()

 1: # 訓練增強版模型
 2: history_aug = model_aug.fit(
 3:     x_train, y_train,
 4:     batch_size=32,
 5:     epochs=30,           # 資料擴增版可以訓練更多 epochs
 6:     validation_split=0.2,
 7:     verbose=1
 8: )
 9: 
10: loss_aug, acc_aug = model_aug.evaluate(x_test, y_test, verbose=0)
11: print(f"\n=== 比較結果 ===")
12: print(f"基礎模型測試準確度：{acc * 100:.2f}%")
13: print(f"資料擴增模型測試準確度：{acc_aug * 100:.2f}%")

7.2. 繪製增強版訓練曲線

2: plot_history(history_aug.history, 'accuracy', 'val_accuracy', 'Accuracy', 'mydata-aug-accuracy.png')
3: print("增強版 Loss 曲線已儲存至 images/mydata-aug-loss.png")

8. 完整流程回顧

步驟 做什麼 為什麼
1. 準備資料 蒐集圖片、分類到資料夾 模型需要有標籤的資料來學習
2. 讀取圖片 用 cv2 讀取、統一大小 模型需要固定大小的輸入
3. 正規化 像素值除以 255 避免梯度爆炸，加速收斂
4. 分割資料 分訓練集和測試集 用測試集檢驗模型有沒有「真正學會」
5. 建模型 CNN（Conv2D + MaxPooling） CNN 特別擅長處理圖片
6. 訓練 model.fit() 讓模型從資料中學習
7. 評估 model.evaluate() + 畫圖 確認模型的好壞
8. 資料擴增 旋轉、翻轉、縮放 用少量圖片也能訓練出好模型

9.1. 作業一：我的 AI 分類器

用你自己拍的照片來訓練一個 CNN 分類器。

  1. 任務說明
    1. 選擇 *兩個類別*（以下任選一組，或自訂）：
      • 手掌張開 vs 握拳
      • 你的文具：鉛筆 vs 橡皮擦
      • 兩種不同的零食包裝
      • 晴天 vs 陰天（用手機拍窗外的天空）
    2. 每個類別拍 至少 30 張 照片（不同角度、不同背景）

    3. 整理成這樣的資料夾結構：

         my_dataset/
   ├── class_a/
   │   ├── img_001.jpg
   │   ├── img_002.jpg
   │   └── ...
   └── class_b/
       ├── img_001.jpg
       ├── img_002.jpg
       └── ...
